# Sails.js Adapter

## Purpose

Use Sails.js runtime as HTTP inbound interface.

## Entrypoints

- `src/interface/HTTP/adapters/sails-js/sails-js.ts`

## Build a Service with Sails.js

1. Keep domain + use cases framework-agnostic.
2. Implement Sails.js framework handlers.
3. Run:

```bash
npm run dev:sails-js
```
