# Derby.js Adapter

## Purpose

Run HTTP interface via Derby.js adapter.

## Entrypoints

- `src/interface/HTTP/adapters/derby-js/derby-js.ts`

## Build a Service with Derby.js

1. Implement domain/use-case/controller operations.
2. Implement Derby-specific handlers.
3. Run:

```bash
npm run dev:derby-js
```
