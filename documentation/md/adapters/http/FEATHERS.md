# Feathers Adapter

## Purpose

Expose API operations through Feathers runtime adapter.

## Entrypoints

- `src/interface/HTTP/adapters/feathers/feathers.ts`

## Build a Service with Feathers

1. Keep service behavior in module use cases.
2. Map Feathers handlers/services to controller operations.
3. Run:

```bash
npm run dev:feathers
```
